## Overview

A desktop application for simulating planar truss structures using the **Finite Element Method (FEM)**, implemented in C#. The user defines nodes, bars and boundary conditions, and the simulator computes displacements, reaction forces and stress in each member.

## Method

Each bar element contributes a local stiffness matrix $\mathbf{k}_e$ assembled into the global system:

$$
\mathbf{K} \mathbf{u} = \mathbf{f}
$$

where $\mathbf{K}$ is the global stiffness matrix, $\mathbf{u}$ the nodal displacement vector, and $\mathbf{f}$ the external force vector. Boundary conditions are applied by eliminating constrained degrees of freedom before solving.

## Features

- Interactive node and bar placement
- Support for pinned and roller boundary conditions
- Real-time visualization of deformed shape and stress distribution
