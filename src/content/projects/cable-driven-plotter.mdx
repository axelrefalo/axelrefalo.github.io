## Overview

This project consists of a parallel robot driven by two cables, designed for spray painting applications. The dynamic model was derived fully analytically, then validated through simulation before being implemented on the physical prototype.

## Approach

The cable-driven architecture offers a lightweight, large-workspace alternative to rigid-link robots. The tension in each cable is controlled to trace arbitrary planar paths while maintaining the end-effector orientation.

Key steps:
- Analytical derivation of the dynamic model (Lagrangian mechanics)
- Validation against numerical simulation
- Implementation and testing on the physical prototype
